# Диаграмма состояний — Document Generation Service

Процесс описывает последовательность шагов, которые пользователь проходит для создания нового документа, заполнив данные в форме на основе шаблона.

![State Diagram](images/state.png)

## 1. Основной процесс генерации документа

Это высокоуровневая схема, отображающая линейный поток от выбора действия до завершения заполнения.

### Состояния процесса

| Состояние | Описание |
| :--- | :--- |
| **Выбор действия** | Пользователь находится в интерфейсе выбора действия и выбирает пункт "Fill out a form". |
| **Подтверждение начала** | Система отображает экран с кнопкой "Continue" для старта процесса заполнения. |
| **Ввод названия** | Пользователь вводит название для нового документа в специальное поле. |
| **Выбор формата** | Пользователь выбирает желаемый формат экспорта будущего документа (например, PDF, DOCX). |
| **Заполнение полей** | Активное состояние: пользователь последовательно заполняет все поля данных в форме. |
